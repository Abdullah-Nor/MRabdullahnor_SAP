# Dev Guide for UI Components

## UI Components

### Layout

Layout is container of the islands of certain screen.
**Location**: `src/layouts/`
**Naming Convention**: {PascalCaseName}Layout.vue

Example: MainLayout.vue, PlainLayout.vue, PrintLayout.vue

```html
<q-layout>
  <q-page-container>
    <router-view />
  </q-page-container>
</q-layout>
```
